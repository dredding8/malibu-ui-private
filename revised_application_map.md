
- **VUE Dashboard** (/)
  - Description: Main application landing page with an overview and navigation.
  - Function: Provides a high-level summary and quick access to major application features.
  - **Key Metrics**
    - Displays critical numbers from the Analytics section.
  - **Quick Links**
    - **Create a new Collection Deck** (-> /decks/new)
    - **Add a new SCC** (-> /sccs/new)

- **SCCs** (/sccs)
  - Description: A dedicated section for the master list of SCCs.
  - Function: View, search, and manage satellite data.
  - **Search and Filter**
    - Prominently displayed controls to search and filter the SCC list.
  - **SCC Table**
    - The master list of SCCs.
  - **Buttons**
    - **Add SCC** (-> /sccs/new)
      - Description: Navigates to the form for adding a new SCC.
    - **Refresh List**
      - Description: Fetches the latest SCC data.

- **Add SCC** (/sccs/new)
  - Description: A dedicated page for adding a new SCC.
  - Function: Create a new satellite catalog entry.
  - **Form Fields**
    - SCC #, Priority, Function, Orbit, Periodicity, Collection Type, Classification.
  - **Buttons**
    - **Save**
      - Description: Saves the new SCC to the master list.
    - **Cancel**
      - Description: Discards changes and returns to the SCC list.

- **Collection Decks** (/decks)
  - Description: A unified section for managing all collection decks.
  - Function: Create new decks and review past and in-progress decks.
  - **Buttons**
    - **Create New Deck** (-> /decks/new/data)
      - Description: Starts the process of creating a new collection deck.
  - **Tabs**
    - **In Progress**
      - Description: A table of collection decks that are ready to continue.
    - **Completed**
      - Description: A table of completed collection decks.
  - **Date Filters**
    - Start Date, End Date, Reset.

- **Create Collection Deck** (/decks/new/*)
  - Description: A multi-page flow for creating a new collection deck.
  - **Step 1: Input Data** (/decks/new/data)
    - **Tasking Window**
    - **TLE Data** (Load from UDL or Import)
    - **Unavailable Sites** (Load from BLUESTAT or Manual Entry)
    - **Buttons**: Next, Cancel
  - **Step 2: Review Parameters** (/decks/new/parameters)
    - **Matching Parameters and Sensor Capacity** (Hard Capacity, Min Duration, Elevation)
    - **Buttons**: Next, Back, Cancel
  - **Step 3: Review Matches** (/decks/new/matches)
    - **Buttons**: Next, Back, Cancel
  - **Step 4: Special Instructions** (/decks/new/instructions)
    - **Buttons**: Finish, Back, Cancel

- **Analytics** (/analytics)
  - Description: Provides data visualization and analysis.
  - Function: Explore trends and metrics.
  - **Controls**
    - **Viewing Dropdown**
    - **Export to Excel Button**
  - **Charts**
    - % of Matching, General Matches, Cumulative Matches (each with a download button).
  - **Audit Log Table**

- **Global Elements**
  - **Logout Button**
    - Consistently available in the application header.
